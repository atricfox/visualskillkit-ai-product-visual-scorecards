# AI Product Visual Scorecards

Practical review scorecards for AI-assisted product visuals and image prompts.

These resources are designed for ecommerce teams, creators, and developers who use AI image tools but still need a clear review process before publishing visual assets.

## Core Rule

Product facts are fixed. The scene is editable.

AI can help with backgrounds, surfaces, crops, lighting, campaign concepts, and layout exploration. It should not quietly change the product a buyer will receive.

## Use Cases

- Reviewing AI-generated ecommerce product photos.
- Checking whether an AI product image changed labels, logos, materials, scale, or claims.
- Debugging image prompts that create attractive images but miss the actual job.
- Creating a shared review protocol for designers, sellers, marketers, and developers.

## What This Is Not

This is not a model benchmark.

It does not claim that one AI image generator is better than another. It is a workflow review protocol: compare the output against the source, score the risk, then publish, repair, or reject.
